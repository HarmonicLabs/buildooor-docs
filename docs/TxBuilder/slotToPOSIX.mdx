# `slotToPOSIX`

```ts
slotToPOSIX(slot: number | bigint): number
```

Converts a slot number to the equivalent POSIX timestamp based on the configured genesis information.

## Parameters

- `slot`: `number | bigint` - The slot number to convert to a POSIX timestamp

## Returns

`number` - The corresponding POSIX timestamp for the given slot number

## Info

This method requires [`GenesisInfos`](../miscellaneous/GenesisInfos) to be configured. If no genesis infos are set, the method throws an `Error`.

You can set the genesis infos using the [`setGenesisInfos`](./setGenesisInfos) method before calling this function.
