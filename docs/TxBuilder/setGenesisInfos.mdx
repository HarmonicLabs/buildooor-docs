# `setGenesisInfos`

```ts
setGenesisInfos(genesisInfos: GenesisInfos): void
```

Sets the genesis information required for slot and time conversions.

## Parameters

- `genesisInfos`: [`GenesisInfos`](../miscellaneous/GenesisInfos) - The genesis information object containing network parameters

## Returns

This method doesn't return a value.

## Info

This method must be called before using time/slot conversion functions like [`posixToSlot`](./posixToSlot) and [`slotToPOSIX`](./slotToPOSIX). The genesis information contains essential network parameters needed for accurate time calculations.
