# `posixToSlot`

```ts
posixToSlot(POSIX: number | bigint): number
```

Converts a POSIX timestamp to the equivalent slot number based on the configured genesis information.

## Parameters

- `POSIX`: `number | bigint` - The POSIX timestamp to convert to a slot number

## Returns

`number` - The corresponding slot number for the given POSIX timestamp

## Info

This method requires [`GenesisInfos`](../miscellaneous/GenesisInfos) to be configured. If no genesis infos are set, the method throws an `Error`.

You can set the genesis infos using the [`setGenesisInfos`](./setGenesisInfos) method before calling this function.
