# `attachMintingValidator`

```ts
attachMintingValidator(
    validator: Script<PlutusScriptType>
): TxBuilderRunner
```

Alias for [`attachValidator`](./attachValidator). This method is provided for semantic clarity when specifically attaching minting policy validators.

## Parameters

- `validator`: The Plutus script to be attached as a minting policy validator

## Returns

Returns a reference to the same `TxBuilderRunner` instance for method chaining.
