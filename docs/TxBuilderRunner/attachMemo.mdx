# `attachMemo`

```ts
attachMemo(
    memo: string
): TxBuilderRunner
```

Adds a metadata message according to [CIP-0020](https://cips.cardano.org/cips/cip20/). This method provides a convenient way to attach human-readable messages to transactions.

## Parameters

- `memo`: The message to be attached as metadata

## Returns

Returns a reference to the same `TxBuilderRunner` instance for method chaining.
