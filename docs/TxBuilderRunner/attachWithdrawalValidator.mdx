# `attachWithdrawalValidator`

```ts
attachWithdrawalValidator(
    validator: Script<PlutusScriptType>
): TxBuilderRunner
```

Alias for [`attachValidator`](./attachValidator). This method is provided for semantic clarity when specifically attaching withdrawal validators.

## Parameters

- `validator`: The Plutus script to be attached as a withdrawal validator

## Returns

Returns a reference to the same `TxBuilderRunner` instance for method chaining.
